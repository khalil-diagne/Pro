# Backend

A PHP and Hack backend project.

## Overview

This repository contains the backend implementation using PHP (87.7%) and Hack (12.3%).

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository
2. Install dependencies
3. Configure your environment
4. Run the application

## Project Structure

- PHP: Primary language for backend logic
- Hack: Additional typing and framework support

## Requirements

- PHP 7.4 or higher
- Composer for dependency management

## Installation

```bash
composer install
```

## Usage

Run the application with:

```bash
php -S localhost:8000
```
